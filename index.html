🏋️ Gym Tracker
Connect Google Sheets for free cross-device sync
1
Create a Google Sheet
Go to sheets.new and create a blank spreadsheet. Name it Gym Tracker.
2
Open Apps Script
In the sheet, click Extensions → Apps Script. Delete any existing code.
3
Paste the backend code
Copy everything from Code.gs and paste it into the editor. Press Ctrl+S.
4
Deploy as a Web App
Click Deploy → New Deployment. Set type to Web App. Set Who has access to Anyone. Click Deploy. Copy the URL.
5
Paste the URL below
Paste your deployment URL and tap Connect.
